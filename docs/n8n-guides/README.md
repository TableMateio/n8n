# n8n Guides

This directory contains practical guides and documentation for working with n8n in the Tax Surplus Recovery System.

## Documents

- [n8n-data-handling.md](./n8n-data-handling.md) - Understanding data flow between nodes and best practices for data handling
- [n8n-bridge-workflow-management.md](./n8n-bridge-workflow-management.md) - Using tools for programmatic workflow management
- [n8n-bridge-usage-guide.md](./n8n-bridge-usage-guide.md) - Guide for using the n8n Bridge with Cursor
- [n8n-credentials-management.md](./n8n-credentials-management.md) - Managing credentials securely in n8n

## Purpose

These guides provide:

- Practical tips for working with n8n
- Solutions to common challenges
- Patterns for handling data and workflows
- Tool usage guides

## When to Use These Guides

- When you need to understand data flow between nodes
- When you're building or modifying workflows programmatically
- When you need to manage credentials or set up secure connections
- When working with the n8n Bridge and Cursor integration
