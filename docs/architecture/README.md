# System Architecture Documentation

This directory contains documentation related to the high-level system architecture of the Tax Surplus Recovery System.

- [DECISION_GUIDE.md](./DECISION_GUIDE.md) - Framework for making architectural decisions

- Configuration approaches
- Design principles
- Implementation patterns
