# Development Process Documentation

This directory contains documentation related to the development process, workflows, and best practices for the Tax Surplus Recovery System.

## Documents

- [CONTRIBUTING.md](./CONTRIBUTING.md) - Guidelines for contributing to the project
- [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md) - Code of conduct for project contributors
- [DEV_WORKFLOW.md](./DEV_WORKFLOW.md) - Development workflow and processes

## Purpose

The development process documentation defines:

- How to contribute to the project
- Development workflows and procedures
- Testing and quality assurance processes
- Code reviews and collaboration guidelines

## When to Use This Documentation

- When you are new to the project and want to contribute
- When you need to understand the development workflow
- When you are preparing a pull request
- When you need to follow the established development process
